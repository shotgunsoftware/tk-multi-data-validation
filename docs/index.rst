ShotGrid Data Validation App
##############################

Overview
========================

The Data Validation App is a customizable workflow tool that allows studios to help control the quality and format of the data created by artists. To first get an understanding of what the app does, and how it performs the validation, see :ref:`using-data-validation-app`.

Once you're ready to start setting up the app, see the step-by-step guide in :ref:`getting-started`.
